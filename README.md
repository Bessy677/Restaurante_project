El presente proyecto consiste en el desarrollo de una aplicacion para la gestion básica de pedidos de un restaurante, utilizando Python 3 como lenguaje de programacion y el framework Kivy para la creacion de la interfaz gráfica. El sistema fue diseñado con el proposito de simular el funcionamiento de un menu digital interactivo, permitiendo visualizar productos y facilitar el proceso de selección de pedidos.

La aplicación incluye un menu compuesto por un minimo de seis productos, los cuales estan organizados y almacenados en un archivo externo menu.json. Este archivo permite manejar la informacion de manera estructurada, separando los datos de la logica del programa. Gracias a este enfoque, los productos pueden ser modificados, agregados o eliminados sin necesidad de cambiar el codigo principal.

Mediante la interfaz desarrollada en Kivy, el usuario puede visualizar los articulos disponibles, incluyendo detalles como nombre, precio y descripcion (segun la informacion registrada en el JSON). El programa procesa estos datos para mostrarlos dinamicamente en pantalla, demostrando el uso de estructuras de datos, lectura de archivos en formato JSON y diseño de interfaces graficas.

Como resultado, el proyecto evidencia la integración de varias competencias de programacion, tales como:

Manejo de archivos externos (JSON)

Programación estructurada en Python

Desarrollo de interfaces graficas

Organización y gestión de informacion digital
